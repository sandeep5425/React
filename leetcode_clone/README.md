**LeetCode clone**

**Project Setup:**

1.Clone this repo.
```
git clone https://github.com/sandeep5425/React.git
```

2.Go inside folder leetcode_clone
```
cd leetcode_clone
```

3.Install all the dependencies
```
npm install
```

4.Run the application.
```
npm run dev
```

5.Open the url in the browser.
```
http://localhost:5173/
```
